# Getting Started

## Compile Example

    $ cd project_templates/c_cpp
    $ mkdir build
    $ cmake ..
    $ make


## Generate Documentation

    $ cd project_templates/c_cpp
    $ mkdir build
    $ cmake ..
    $ make documentation


## Run Example

    $ cd project_templates/c_cpp/build
    $ src/example/example -h
    $ src/example/example -f foo -l bar
